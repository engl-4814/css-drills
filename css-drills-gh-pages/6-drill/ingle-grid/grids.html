Type of Trees

Below is a list of different types of trees

Pine
  • These are evergreen trees.
  • Pine cones are produced from this tree
  • This is a sap producing tree.
Oak
  • This is a deciduous tree.
  • Acorns are produced from this tree.
Maple
  • "Helicopter seeds" are actually maple seeds.
  • Maple trees produce syrup.
  • Maple trees can live hundreds of years.
